# Change Log

All notable changes to the "codrag-vscode" extension will be documented in this file.

## [0.1.0] - 2025-02-09

### Added
- **Semantic Search**: Search your codebase using natural language with local-first retrieval.
- **Context Assembly**: Assemble prompt-ready context for LLMs with token estimation.
- **Trace Index (Pro)**: Visualize and navigate code structure with graph-based analysis.
- **Project Management**: Manage multiple projects via the sidebar (Add, Remove, Rebuild).
- **File Tree**: Browse indexed files and view coverage status.
- **Index Status**: Monitor index freshness, chunk counts, and build status.
- **Licensing**: Activate/Deactivate licenses directly within VS Code.
- **Theme Integration**: Seamless integration with your current VS Code theme.
- **Intelligent Project Selection**: Automatically selects the active project based on the open file.
- **Keybindings**: Quick access to core features (Search: `Ctrl+Alt+S`, Context: `Ctrl+Alt+C`, Build: `Ctrl+Alt+B`, Trace: `Ctrl+Alt+T`).
